This sets the environment for using the cross tools.  It must
be run from the ~/sros directory, but then all environment
variables are ready to be used in other directories.

The following Environment Variables are set:

SROS_ROOT             #  Root Directory of ~/sros
BUILDROOT_BASE        #  Full Path to Buildroot
KERNEL_BASE           #  Full Path to Kernel
TOOLCHAIN_DIR         #  Full Path to Cross Tools
TOOLCHAIN_PREFIX      #  Prefix of powerpc-linux-
PATH                  #  Path to Cross Tools included.
